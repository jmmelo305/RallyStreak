# Game Design Document

## Game Title
Rally Streak

## Game Overview

A game based on the real life sport Tennis. The game consists of seeing how many time you can consecutively rally with the npc. As your score goes up, so does the difficulty.

## Core Mechanics

1. Racquet Movement up / down / left / right
2. Left click to hit the ball
3. Shot Placement
- W = Lob Shot
- A = Left Corner Shot
- S = Drop Shot
- D = Right Corner Shot
4. Hidden Konami Code Easter Egg

## How To Run: Prerequisites
- Use this link: https://jmmelo305.github.io/RallyStreak/
- Functioning Keyboard & Mouse.

## Steps
1. Click to start
2. Left click to serve the ball.
3. Hold W / A / S / D to determine the placement of your shots
4. Aim at the ball with the racquet and left click to swing.
5. 3 lives, if you miss a ball you lose a life.
6. Press "ESC" at any moment to return to the Title Screen.

## Win Conditions

Keep up the rally streak against the NPC, reach new highscores! 

## Lose Conditions

If you miss the ball 3 times you lose.
